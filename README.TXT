!!!Warning!!!
Script overwrites current ~/.config configs of fastfetch, icewm, kitty, nvim

Script installing:
    1. uncomment needed optional lines in "./install.sh"
    2. execute "./install.sh"

After-script installing:
    1. delete this git-cloned repo and reboot

Keybinds:
    TABS / WINDOWS
    - ALT + a / s / d / e - goes to the first / second / third / fourth tab
    - ALT + SUPER + a / s / d / e - moving current window to the first / second / third / fourth tab
    - ALT + j - moving to the next window (cycling) of the current tab
    - ALT + w - show list of windows
    - ALT + c - killing current window
    - ALT + x - maximize / minimize current window

    APPS
    - ALT + t - opening terminal
    - ALT + b - opening browser
    - ALT + f - opening file manager
    - ALT + m - opening app menu
